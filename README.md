## ğŸƒâ€â™‚ï¸ Getting Started

### Prerequisites
-   Node.js (v18+)
-   Java JDK 17+
-   Maven

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Gopika-Janarthanan/chef-connect.git
    cd chef-connect
    ```

2.  **Start the Backend**
    ```bash
    cd backend
    mvn spring-boot:run
    ```
    *The backend runs on `http://localhost:8080`.*

3.  **Start the Frontend** (in a new terminal)
    ```bash
    # Install dependencies
    npm install

    # Run development server
    npm run dev
    ```
    *The frontend runs on `http://localhost:5173`.*
